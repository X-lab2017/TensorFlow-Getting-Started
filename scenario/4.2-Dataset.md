TensorFlow已经内置了数据集，因此无需手动下载。
```
from tensorflow.examples.tutorials.mnist import input_data
mnist = input_data.read_data_sets("MNIST_data/", one_hot=False)
```
请注意，我们使用值为False的one_hot参数。 这意味着标签将被读取为整数值而不是一个热编码向量。
